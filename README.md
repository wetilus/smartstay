# 🏨 SmartStay

SmartStay is a full-stack Hotel Management System built to simplify hotel operations such as room management, bookings, customer management, and administrative tasks.

The project is being developed as a portfolio project to demonstrate real-world full-stack development using modern web technologies.

---

## 🚀 Features

### Authentication
- User Registration
- User Login
- JWT Authentication
- Role-Based Access Control (Admin, Receptionist, Customer)

### Room Management
- Add Rooms
- Update Room Details
- Delete Rooms
- View Room Availability

### Booking System
- Book Rooms
- Cancel Bookings
- Booking History
- Check-in & Check-out

### Dashboard
- Total Rooms
- Available Rooms
- Active Bookings
- Revenue Overview

### Future Features
- AI-powered Review Summarization
- Smart Room Recommendations
- Email Notifications
- Online Payments
- Analytics Dashboard
